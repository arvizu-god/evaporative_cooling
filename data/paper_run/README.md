# Canonical paper dataset

Base (`*.json`) and post-processed thermodynamic (`*_thermo.json`) outputs for
the single simulation session behind the figures in the accompanying paper.

The normalized siblings (`*_norm.json`) are **not** committed — they are
regenerated by `reproduce_figures.py`, which is also what produces the figures
in `../figures/`. Regenerating them and getting byte-identical figures is the
reproducibility check.
